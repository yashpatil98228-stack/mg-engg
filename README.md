# 🎓 SSPSIT College Management System

A high-end, production-ready college management system for **Smt. Sharchchandrika Suresh Patil Institute of Technology (SSPSIT), Chopda**. This platform provides a seamless EXPERIENCE for both students and faculty with state-of-the-art UI/UX.

## 💻 Tech Stack
- **React 18** (Modern functional components)
- **Tailwind CSS 3** (Premium styling and glassmorphism)
- **Lucide React** (Professional iconography)
- **Vite** (Optimized build system)

## ✨ Core Features

### 🔹 Student Interface
- **Unified Dashboard**: Real-time attendance circles and quick-access grids.
- **Attendance Hub**: Deep-dive into lecture-wise participation records.
- **Document Center**: Download notes, assignments, and project specs.
- **Result tracker**: Comparative analysis across all semesters and CA exams.
- **Digital Schedule**: interactive daily timetable with room numbering.

### 🔸 Teacher Interface
- **Attendance Manager**: Streamlined batch-wise attendance marking with WhatsApp/SMS simulation.
- **Resource Uploader**: Drag-and-drop file management for academic materials.
- **Student Registration**: Integrated digital enrollment system.
- **Admin Stats**: Dynamic cards showing faculty-wide statistics.

## 🛠 Setup for Developers
1. Ensure **Node.js** is installed.
2. Run `npm install` to download dependencies.
3. Start the dev server with `npm run dev`.
